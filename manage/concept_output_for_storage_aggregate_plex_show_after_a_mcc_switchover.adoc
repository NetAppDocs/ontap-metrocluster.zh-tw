= 儲存Aggregate plex show命令的輸出在MetroCluster 執行完功能介紹後仍不確定
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當您在MetroCluster 執行完還原切換後執行儲存Aggregate plex show命令時、切換根Aggregate的plex0狀態會不確定、並顯示為故障。在此期間、切換根目錄不會更新。此叢的實際狀態只能在MetroCluster 還原階段之後決定。
