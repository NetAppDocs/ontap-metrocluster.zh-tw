= 使用MetroCluster 陣列LUN在組態中安裝及佈線不需執行的元件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要使用MetroCluster 陣列LUN來設定Isfing組態、您必須將儲存控制器纜線連接至FC交換器、然後將ISL纜線連接至站台。此外、您必須將儲存陣列纜線連接至FC交換器。
