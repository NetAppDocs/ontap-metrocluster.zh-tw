= 將ISL佈線至MetroCluster 各個不景點
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您必須透過光纖交換器間連結（ISL）連接每個站台的FC交換器、以形成連接MetroCluster 到該元件的交換器架構。

.關於這項工作
這兩種交換器架構都必須完成。

.步驟
. 使用對應於組態和交換器模型的表格中的纜線、將每個站台的FC交換器連接至所有ISL。
