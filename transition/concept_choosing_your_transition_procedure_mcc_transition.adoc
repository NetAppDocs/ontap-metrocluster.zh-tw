= 選擇您的轉換程序
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您必須根據現有MetroCluster 的不規則功能（NetApp）組態、選擇轉換程序。

轉換程序會以IP交換器網路取代後端FC交換器網路或FC-VI連線。確切的程序取決於您的初始組態。

在轉換程序結束時、會淘汰原始平台和FC交換器（如果有）。
