= 在MetroCluster 不提供的情況下設定功能ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您必須在MetroCluster 不完整的功能組態中設定每個節點ONTAP 、包括節點層級組態、以及將節點組態設定為兩個站台。您也必須在MetroCluster 這兩個站台之間實作「不二」關係。
