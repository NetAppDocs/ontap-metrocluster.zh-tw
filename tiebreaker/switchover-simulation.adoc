= 使用MetroCluster Tiebreaker 模擬切換
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
從MetroCluster Tiebreaker 1.7 開始，您可以模擬切換來測試 Tiebreaker 切換功能。

.關於這項工作
* 這 `monitor switchover-simulate`該命令僅在 Tiebreaker 1.7 或更高版本中受支援。


.步驟
. 查看決勝局監控狀態：
`monitor show -status`

. 執行切換模擬：
+
`monitor switchover-simulate -monitor-name <monitor_name> -cluster <cluster_name>`

+
該命令成功執行後，將返回以下輸出：
